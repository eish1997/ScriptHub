# creative-production-runtime CodeCanvas 架构画布

本目录由 `npm run canvas:init` 生成，含 **项目分析运营图**（非反向指导开发的蓝图）。

- **用途**：在 Portfolio 中理解「这个项目做什么、入口在哪」
- **更新**：代码变更后在 Cursor 对 Agent 说「更新 creative-production-runtime 的运营图」
- **DevTask**：在 Hub 看板创建，填写 `targetProjectId=creative-production-runtime`
- **正式依据**：`published.operation-map.json`（分析视图，可随代码演进刷新）
